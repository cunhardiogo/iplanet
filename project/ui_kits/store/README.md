# iPlanet Store — UI Kit

A pixel-faithful recreation of the iPlanet store landing page, modeled on `apple.com/br/store`.

## Files
- `index.html` — interactive landing page mock (entry point)
- `TopNav.jsx` — translucent blurred top navigation
- `CategoryRail.jsx` — product silhouettes nav
- `SectionTitle.jsx` — coral headline + grey subhead pattern
- `ProductCard.jsx` — white product card with NOVO flag, title, price
- `HeroCard.jsx` — dark/wash full-bleed hero card
- `FeatureRow.jsx` — coral-icon feature tiles
- `QuickLinks.jsx` — outlined pill utility nav
- `Footer.jsx` — long legal-text footer
- `data.js` — product fixtures

## How to view
Open `index.html`.

## Notes
- Components are simple, mostly cosmetic recreations.
- Product imagery is represented by colored gradient placeholders (no real product photos shipped). Drop real cut-outs into `/assets/products/` and reference by name.
- All text is in pt-BR.
